# ECM 등가회로 모델 연구포털

**군산대학교 기계공학부 | Plett (2015) BMS Vol.1 Chapter 02**

## 🚀 로컬 실행

```bash
pip install -r requirements.txt
streamlit run app.py
```

## ☁️ Streamlit Cloud 배포

1. GitHub에 이 폴더를 업로드
2. [share.streamlit.io](https://share.streamlit.io) 접속
3. New app → GitHub 연결 → Main file: `app.py`
4. Deploy!

## 📦 포함 기능

- 📖 **주제별 학습** (8개 섹션, 5탭 구성: 개념·수식·그래프·비교·참고문헌)
- 🎯 **핵심 포인트** — 4개 카드 형식의 핵심 요약
- 📝 **상세 내용** — 심화 설명 텍스트
- 📊 **AI 그래프** — Plotly 인터랙티브 그래프 (소제목별)
- 🛠 **인터랙티브 실습** 3종 (실제 작동):
  - SoC 실시간 시뮬레이터
  - 나이키스트 플롯 단계별 학습
  - ESC 모델 단계별 구성
- 🧮 **파라미터 계산기** + Chart.js 시뮬레이터
- 📝 **이해도 퀴즈** 10문제 (4지선다 + 해설)
- 🌤 **군산 날씨** 실시간 API
